# .vimrcのシンボリックリンクの作成

カレントディレクトリが本リポジトリのルートのときに
```
$ ln -s .vimrc ~/.vimrc
```


# チートシート

## 暗記したいもの

 - `a`
 - `A`
 - `I`
 - `o`
 - `O`
 - `y`
 - `Y`
 - `p`
 - `P`

## 重要だけど時々しか使わないので暗記しなくていいもの

## もう暗記してしまったもの
 - `i`
 - `:q`, `:w`, `:wq`, `ZZ`
 - `h`, `j`, `k`, `l`
 - `C-[`, `Esc`
 - `dd`
 - `u`, `C-r`
