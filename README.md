# 初期設定

1. MacVim インストール

   ```
   brew install macvim
   ```

1. .vimrc のシンボリックリンク作成

   ```
   $ ln -s [本リポジトリのルートデ ィレクトリ絶対パス]/.vimrc ~/.vimrc
   ```
